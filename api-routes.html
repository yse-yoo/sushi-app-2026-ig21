要対応箇所

FIX 01

categoryService.js — loadCategories()

fetch(url, options) と response.json() がコメントアウトされており、response 変数が未定義のまま参照されるため実行時エラーになる。

FIX 02

orderService.js — submitOrder()

URL: ${apiBaseUrl} → ${apiBaseUrl}/api/order/add
method: '' → 'POST'
headers: { 'Content-Type': '' } → { 'Content-Type': 'application/json' }
body: JSON.stringify() → JSON.stringify({ product_id: productId, quantity, visit_id: Number(visitId) })
INFO 03

/api/seat/available ・ /api/seat/find

バックエンドにルートは存在するが、フロントエンドの対応サービス関数がない（現時点では未使用）。
